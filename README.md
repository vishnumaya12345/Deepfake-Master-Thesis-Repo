\# Master Thesis – DeepfakeBench Experiments (Code + Results)



This repository contains the Colab notebooks, scripts, and exported results used for my master’s thesis on deepfake detection using DeepfakeBench.



\## What is included

\- `notebooks/` : 27 Colab notebooks used for data processing, inference, and result generation

\- `results/`   : exported tables/plots/logs (no large videos)

\- `src/`       : helper scripts (if any)

\- `environment/`: requirements or setup files

\- `data/`      : dataset instructions only (datasets are NOT included)



\## Important note about datasets

Datasets contain real faces and are not uploaded to this public repository for privacy and licensing reasons.

Instructions for dataset folder structure are provided in `data/README.md`.
