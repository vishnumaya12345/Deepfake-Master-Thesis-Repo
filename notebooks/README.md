\# Notebooks index



\## Naming convention

\- Files containing `CelebDF` = runs on Celeb-DF dataset

\- Files containing `final` = runs on FF++ (C40) dataset

\- Files containing `1 src`, `5 src`, `10 src` = runs on custom identity-swap datasets with 1/5/10 source images

\- Model names in filenames indicate the detector/architecture used (e.g., Xception, F3Net, FFD, Capsule, EfficientNet)



\## Suggested reading / execution order (high level)

1\) Dataset preparation / frame extraction notebooks (if any)

2\) Detector inference notebooks (per dataset/model)

3\) Result aggregation / plots notebooks (if any)




\## Notes

\- Datasets are NOT stored in this repository (privacy/licensing). See `../data/README.md`.
